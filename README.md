## 📋 プロジェクト概要

**KORU 学習支援・運用統合アプリ（Web）**

Pythonを用いたバックエンドとAI機能を中核に、英語学習体験と運用業務を一体化する包括的な学習支援プロダクト

### 主な機能

### ① 予約管理

- レッスン予約管理（**Google Meet / Zoom 自動連携**）
- 予約完了時にミーティングリンクを自動生成・通知
- 生徒ダッシュボードに「次のレッスン」を常時表示

### ② Conversation Class 管理

- Conversation Class の予約管理（**Zoom 自動連携**）
- **チューター向け管理画面**
    - 対応可能な日時の登録
    - 生徒側予約画面へ自動反映
- 生徒・チューター双方のスケジュール可視化

### ③ 教材管理

- Notion連携による教材閲覧・管理
- レッスン後のまとめノート共有

### ④ 単語帳

- 内製単語帳機能
- Quizlet連携（将来的に検討）
- レッスン中に「言えなかった単語」を自動保存

### ⑤ 学習コミュニケーション

- Webアプリ内メッセージ機能
- 公式LINEからの脱却
- 学習連絡・簡易フィードバックを一元化

### ⑥ レポート・カリキュラム管理

- レッスンレポート閲覧
- クールごとの学習カリキュラム管理

### ⑦ 学習目標・達成度の可視化

- 目標設定と進捗表示
- 日次／週次の学習タスク完了状況を可視化

### ⑧ AI学習支援機能

- **英作文のAI添削**
- **作文・会話ログからの語彙自動抽出**
- **AI音声生成による発音確認**
- 学習履歴と連動した復習体験の提供

## 🎯 目的・ゴール

### ターゲット

- KORUの生徒（継続的な学習が必要な英語学習者）

### 課題

- 予約管理・変更対応の運用負荷が高い（レッスン／TenTalk）
- 学習が点在し、進捗や成果が把握しづらい
- 公式LINE中心の運用によるUX低下（動作の重さ／ファイル不安定）

### ゴール

- 学習・予約・コミュニケーションを一つの体験に統合
- AIを活用し、復習・定着までを自動化・効率化

### 成功指標（KPI）

- 予約変更に要する工数の削減率
- 週次アクティブ率（WAU）
- 学習タスク完了率／継続率
